# MasterDummyData

## How to Run

1. **Start the services using Docker:**

   ```bash
   docker-compose up -d
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the application:**

   ```bash
   npm run start
   ```

4. **Test the API:**

   ```bash
   curl -X GET http://localhost:3000/api/brokerage/segment
   ```
